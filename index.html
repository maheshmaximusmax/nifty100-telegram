🔑 GitHub Configuration (Required for Control Panel)
To control your bot from this dashboard, enter your GitHub details below. These are saved only in your browser — never uploaded anywhere.
How to create your GitHub Token (PAT):
1. Go to github.com → tap your profile picture (top right) → Settings
2. Scroll all the way down on left sidebar → tap Developer settings
3. Tap Personal access tokens → Tokens (classic)
4. Tap Generate new token (classic)
5. Give it a name like "nifty dashboard"
6. Tick these boxes: repo and workflow
7. Tap Generate token at bottom → Copy the token shown (starts with ghp_)
8. Paste it in the field above
⚠️ GitHub shows this token only ONCE. Copy it immediately.
🤖 Update Telegram Bot / Chat ID
Your BOT_TOKEN and CHAT_ID are stored securely in GitHub Secrets. To change them:
GitHub Repo → Settings → Secrets and variables → Actions → tap ✏️ pencil next to the secret name
📝 How the Schedule Update Works
When you tap "Apply New Time" on the Control tab, this dashboard calls the GitHub API to automatically update the nifty100.yml file with the new cron time. No manual editing needed — it happens in seconds.
