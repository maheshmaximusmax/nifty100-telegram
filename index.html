📨 Telegram Recipients
Add any Telegram user ID, group ID, or channel username (like @mychannel). The bot will send the CSV to ALL of them every day. These are saved in GitHub Secrets as RECIPIENTS.
Examples: 912345678 (user ID)   -1001234567890 (group/channel ID)   @mychannel (public channel)
How to get a Channel ID:
1. Add your bot as an Admin to the channel
2. Forward any message from the channel to @userinfobot
3. It will show the channel ID (starts with -100...)
4. Add that ID here
How the Bot Token Reads Recipients
The script reads the RECIPIENTS GitHub Secret which is a JSON array like:
["912345678", "@mychannel", "-1001234567890"]

When you save from this page, it automatically updates that secret via GitHub API.
